    *   **特点：** 这类插件是专门为了导出对话而生的。安装后，它会在你的 AI 对话页面（如 ChatGPT、Claude、Gemini 等）的右下角或侧边栏生成一个悬浮按钮。
    *   **功能：** 点击即可选择导出为 **Markdown (.md)**、PDF 或是 PNG 图片。它会自动保留对话的角色标签（User / Assistant），并完美兼容代码块和表格。
*   **Save ChatGPT / AI To Markdown**
    *   **特点：** 轻量级，专注于 Markdown 格式。
    *   **功能：** 一键复制整篇对话的 Markdown 源码到剪贴板，或者直接下载为 `.md` 文件。拿到文本后，你可以直接秒贴到 WordPress 的 Markdown 编辑器里。

### 2. 多功能知识库/剪藏插件（适合写博客、做笔记）

如果您平时经常写博客，用这类“剪藏”插件会更顺手，因为它们不仅能导出，还能直接同步到你的笔记软件里：

*   **MarkDownload - Markdown Web Clipper**
    *   **特点：** 一个开源的网页转 Markdown 剪藏利器。
    *   **功能：** 它不仅能识别 AI 的对话界面，还能把任何网页一键转换成干净、无广告的 Markdown 文本。支持自定义导出的模板（比如自动加上日期、网址等）。
*   **SimpRead（简悦）**
    *   **特点：** 国内开发者做的一款极强的内容提取工具。
    *   **功能：** 开启后可以把 AI 对话界面变成完美的“阅读模式”，然后一键导出为标准 Markdown，甚至可以直接对接发布到各种自媒体平台或语雀、Notion 等知识库。
